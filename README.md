Student Management System (Java)

 Overview
A console-based Student Management System built using Core Java.
Implements Object-Oriented Programming principles and file-based storage.

 Features
- Add Student
- Edit Student
- Delete Student
- Search Student
- View All Students
- Persistent file storage (students.txt)

 Technologies Used
- Core Java
- OOP Concepts
- File Handling
- Collections Framework
  How to Run
1. Compile all Java files
2. Run Main.java
3. Use the menu to manage students
